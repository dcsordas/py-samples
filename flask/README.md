Flask code samples
=

REST API examples for Flask.


Installation
-

`pip install -r requirements.txt`


Usage
-

`python -m <module> --help`

Running tests:

`python -m <module>.test`

or:

`python -m <module> --test`


api_basic_auth
-

With basic HTTP authentication.


api_simple
-

Using module level functions only for end points.
